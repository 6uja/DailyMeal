# DailyMeal 🍽️

Flutter 기반 식단·체중·운동 관리 애플리케이션

## 프로젝트 소개

DailyMeal은 사용자가 하루 식단, 체중, 물 섭취량, 운동 기록을 날짜별로 관리할 수 있는 건강 관리 앱입니다.

단순히 식단만 기록하는 것이 아니라 칼로리 계산, PFC 자동 계산, 체중 변화 그래프, 캘린더 기반 기록 관리, SQLite 로컬 저장 기능을 포함하여 실제 식단관리 앱처럼 사용할 수 있도록 구현했습니다.

스토어 배포 없이 Android Emulator 또는 Flutter 시뮬레이터에서 실행할 수 있습니다.

---

## 주요 기능

### 1. 식단 기록

* 음식 이름 입력
* 아침, 점심, 저녁, 간식 구분
* 칼로리 입력
* 메모 작성
* 식단 이미지 선택
* 등록된 식단 삭제

### 2. 칼로리 계산

* 날짜별 총 섭취 칼로리 자동 계산
* 목표 칼로리와 비교
* ProgressBar를 활용한 시각적 표시
* 운동 소모 칼로리 함께 표시

### 3. PFC 자동 계산

* 음식 이름과 칼로리를 기준으로 단백질, 지방, 탄수화물 자동 계산
* 닭가슴살, 계란, 고구마, 밥, 샐러드, 요거트 등 기본 음식 데이터 반영
* 등록되지 않은 음식은 비율 계산 방식으로 자동 추정

### 4. 체중 기록

* 날짜별 체중 입력
* 체중 메모 작성
* 최근 체중 확인
* 목표 체중과 비교

### 5. 체중 변화 그래프

* 입력된 체중 데이터를 기반으로 선 그래프 표시
* 2개 이상의 체중 기록이 있을 때 변화 추이 확인 가능
* fl_chart 패키지를 활용한 그래프 UI 구현

### 6. 물 섭취 기록

* 날짜별 물 섭취량 기록
* 하루 목표 8잔 기준 관리
* 물 섭취 추가 기능
* 물 섭취 초기화 기능

### 7. 운동 기록

* 운동 이름 입력
* 운동 시간 입력
* 소모 칼로리 입력
* 운동 메모 작성
* 날짜별 운동 기록 관리

### 8. 캘린더 기반 기록 관리

* 날짜 선택 기능
* 선택한 날짜의 식단, 운동, 물 섭취 기록 확인
* table_calendar 패키지를 활용한 캘린더 UI 구현

### 9. SQLite 로컬 저장

* 식단 기록 저장
* 체중 기록 저장
* 운동 기록 저장
* 물 섭취 기록 저장
* 앱을 종료해도 데이터 유지

### 10. 이미지 선택 기능

* image_picker 패키지를 활용하여 갤러리 이미지 선택
* 선택한 이미지를 식단 기록과 함께 저장
* 홈 화면 식단 목록에서 이미지 확인 가능

## 화면 구성

### 홈 화면

* 선택된 날짜 표시
* 오늘 섭취 칼로리 표시
* 목표 칼로리 진행률 표시
* 운동 소모 칼로리 표시
* PFC 요약 표시
* 최근 체중 표시
* 물 섭취량 표시
* 식단 목록 확인 및 삭제

### 기록 추가 화면

* 식단 기록 추가
* 체중 기록 추가
* 운동 기록 추가
* SegmentedButton을 사용하여 입력 항목 전환
* 식단 이미지 선택 기능 제공

### 캘린더 화면

* 날짜 선택
* 선택한 날짜의 기록 요약 확인
* 식단 개수, 운동 개수, 물 섭취량 표시

### 통계 화면

* 총 칼로리 확인
* 운동 기록 개수 확인
* PFC 총합 확인
* 물 섭취량 확인
* 체중 변화 그래프 확인

## 데이터베이스 구조

### meals 테이블

식단 기록을 저장합니다.

| 컬럼명       | 설명       |
| --------- | -------- |
| id        | 식단 고유 번호 |
| date      | 기록 날짜    |
| name      | 음식 이름    |
| type      | 식사 종류    |
| calories  | 칼로리      |
| protein   | 단백질      |
| fat       | 지방       |
| carb      | 탄수화물     |
| memo      | 메모       |
| imagePath | 이미지 경로   |

### weights 테이블

체중 기록을 저장합니다.

| 컬럼명    | 설명          |
| ------ | ----------- |
| id     | 체중 기록 고유 번호 |
| date   | 기록 날짜       |
| weight | 체중          |
| memo   | 메모          |

### exercises 테이블

운동 기록을 저장합니다.

| 컬럼명      | 설명          |
| -------- | ----------- |
| id       | 운동 기록 고유 번호 |
| date     | 기록 날짜       |
| name     | 운동 이름       |
| minutes  | 운동 시간       |
| calories | 소모 칼로리      |
| memo     | 메모          |

### water 테이블

물 섭취량을 저장합니다.

| 컬럼명   | 설명       |
| ----- | -------- |
| date  | 기록 날짜    |
| count | 물 섭취 잔 수 |

## 실행 전 확인 사항

이미지 선택 기능을 테스트하려면 에뮬레이터 또는 실제 기기에 갤러리 이미지가 있어야 합니다.

SQLite는 앱 내부 저장소에 자동으로 생성되므로 별도의 데이터베이스 파일을 직접 만들 필요가 없습니다.
